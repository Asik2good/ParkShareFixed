# ParkShare 🚗
MVP системы краткосрочной аренды частных парковочных мест в Алматы.

## О проекте
Сервис позволяет владельцам парковочных мест монетизировать пустующие локации, а водителям — находить безопасную парковку в загруженных районах города.

### Стек технологий:
- **Backend:** Node.js
- **Database:** SQLite
- **Architecture:** Система ролей (Admin, Driver, Owner)
- **Financial Logic:** Автоматический расчет комиссии 15% и выплат владельцам.
